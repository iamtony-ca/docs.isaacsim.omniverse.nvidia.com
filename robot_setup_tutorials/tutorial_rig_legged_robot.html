Tutorial 13: Rigging a Legged Robot for a Locomotion Policy#

이 튜토리얼에서는 이동 정책이 지정한 구성에 맞게 다족 로봇을 리깅하는 방법을 설명합니다. Isaac Sim의 Policy Controller Class는 Isaac Sim에서 추론을 위한 런타임 로봇 리깅을 이미 처리하므로, 이 튜토리얼은 ROS 같은 외부 프로세스에서 로봇 정책을 실행하려는 경우에만 관련이 있습니다.

Learning Objectives#

이 튜토리얼에서는 H1 인간형 로봇을 H1 평지 이동 정책이 지정한 구성에 맞게 리깅하는 과정을 진행합니다.

  1. 초기 로봇 위치 설정

  2. 조인트 구성 설정

  3. 조인트 구성 확인

Note

H1 평지 정책 환경 정의 파일은 여기에서 사용 가능합니다.

Setting the Initial Robot Position#

로봇의 초기 조인트 위치는 환경 정의 파일의 robot:init_state:joint_pos 섹션에 지정됩니다. 조인트 이름은 .* 와일드카드를 사용하여 지정됩니다.

 1robot:
 2  init_state:
 3    joint_pos:
 4      .*_hip_yaw: 0.0
 5      .*_hip_roll: 0.0
 6      .*_hip_pitch: -0.28
 7      .*_knee: 0.79
 8      .*_ankle: -0.52
 9      torso: 0.0
10      .*_shoulder_pitch: 0.28
11      .*_shoulder_roll: 0.0
12      .*_shoulder_yaw: 0.0
13      .*_elbow: 0.52
14    joint_vel:
15      .*: 0.0

Note

조인트 위치는 라디안으로 지정되지만 USD 조인트 위치는 도(degree)로 지정됩니다.

로봇의 초기 상태를 저장하려면:

  1. 콘텐츠 브라우저의 Isaac Sim/Robots/Unitree/H1에서 h1.usd 파일을 엽니다.

  2. 스테이지 우측 상단 모서리에서 funnel 아이콘을 선택하고 Physics Joints를 클릭하여 조인트 목록을 필터링합니다.

  3. 첫 번째 조인트(left_hip_yaw)를 좌클릭한 다음 마지막 조인트(right_elbow)를 Shift+좌클릭하여 모든 조인트를 선택합니다.

  4. 선택된 조인트를 우클릭하고 Add > Physics > Joint State Angular를 클릭하여 조인트에 Joint State API 속성을 만듭니다.

  5. 선택된 조인트를 우클릭하고 Add > Physics > Angular drive를 클릭하여 조인트에 조인트 드라이브 API 속성을 만듭니다.

Note

Joint State Angular API는 조인트 위치와 속도를 보고하고, Angular drive API는 조인트를 구동합니다. 조인트에 이미 Joint State Angular API 또는 Angular drive API가 있으면 이전 두 단계를 건너뛸 수 있습니다.

  1. 각 활성 조인트에 대해 joint_pos와 joint_vel 값을 라디안에서 도(degree)로 변환합니다.

  2. 변경하려는 조인트를 좌클릭합니다.

  3. Property 패널에서 Target Position 속성으로 스크롤합니다.

  4. Target Position 속성을 환경 정의 파일의 joint_pos 속성에서 변환된 값으로 설정합니다.

  5. Target Velocity 속성을 환경 정의 파일의 joint_vel 속성에서 변환된 값으로 설정합니다.

  6. 각 활성 조인트에 대해 이전 단계를 반복합니다.

  7. Play를 누릅니다.

Note

Newton을 사용할 때 /h1/Joints/torso에 역방향 조인트가 지원되지 않아 물리 초기화가 실패할 수 있습니다. 이런 경우 torso 조인트를 선택합니다. Property 패널에서 Physics > Joint로 이동하여 Body 0이 /h1/torso_link이고 Body 1이 /h1/pelvis가 되도록 조인트 바디를 교체합니다.

  1. 로봇이 환경 정의 파일에 지정된 초기 위치로 이동하는지 확인합니다. 시뮬레이션 시작 시 로봇이 초기 위치에서 시작하도록 하려면 Joint State API에 데이터를 저장합니다.

  2. 로봇이 무한정 떨어지는 것을 방지하려면 /h1/torso_link를 우클릭하고 Create > Physics > Joint > Fixed Joint를 선택하여 로봇과 월드 사이에 고정 조인트를 추가합니다.

로봇 포즈를 저장하려면:

  1. 스테이지 왼쪽 상단 모서리에서 Edit > Preferences를 클릭합니다.

  2. Preferences 창에서 왼쪽 사이드바의 Physics 탭을 클릭합니다.

  3. Reset Simulation on Stop을 해제합니다.

  4. 시뮬레이션을 실행하고 로봇이 원하는 초기 포즈에 도달하면 멈춥니다. 리셋 후에도 포즈가 저장되도록 이 단계를 한 번 더 반복합니다.

  5. 로봇과 월드 사이의 고정 조인트를 삭제합니다.

  6. Ctrl+S를 눌러 USD 파일을 저장합니다.

  7. Reset Simulation on Stop을 다시 선택합니다.

Setting the Joint Configuration#

정책의 로봇 구성과 일치하도록 조인트 구성을 설정합니다. 이것은 USD 파일에 저장된 값과 다를 수 있습니다. 조인트 드라이브 구성은 환경 정의 파일의 scene:robot:actuators 섹션에 지정됩니다.

다음 스니펫은 H1 로봇 다리에 대한 액추에이터 구성을 보여줍니다.

 1actuators:k
 2  legs:
 3    class_type: omni.isaac.lab.actuators.actuator_pd:ImplicitActuator
 4    joint_names_expr:
 5    - .*_hip_yaw
 6    - .*_hip_roll
 7    - .*_hip_pitch
 8    - .*_knee
 9    - torso
10    effort_limit: 300
11    velocity_limit: 100.0
12    stiffness:
13      .*_hip_yaw: 150.0
14      .*_hip_roll: 150.0
15      .*_hip_pitch: 200.0
16      .*_knee: 200.0
17      torso: 200.0
18    damping:
19      .*_hip_yaw: 5.0
20      .*_hip_roll: 5.0
21      .*_hip_pitch: 5.0
22      .*_knee: 5.0
23      torso: 5.0
24    armature: null
25    friction: null

joint_names_expr는 액추에이터에 의해 제어될 조인트 이름 목록입니다. class_type은 액추에이터 유형입니다. effort_limit는 조인트에 가할 수 있는 최대 힘입니다. velocity_limit는 조인트에 가할 수 있는 최대 속도입니다. stiffness는 조인트 강성을 정의합니다. damping은 조인트 감쇠를 정의합니다. armature는 조인트 아마추어를 정의하고 friction은 조인트 마찰을 정의합니다.

조인트 구성을 설정하려면:

  1. left_hip_yaw와 같은 조인트를 좌클릭합니다.

  2. Property 패널에서 Joint Drive 속성으로 스크롤하고 환경 정의 파일에 지정된 값으로 stiffness와 damping을 설정합니다.

Note

stiffness와 damping을 도(degree) 기반 단위로 변환하는 것을 잊지 마세요.

USD 파일의 stiffness는 \(\frac{kg \cdot m^2}{deg \cdot s^2}\) 단위이고 damping은 \(\frac{kg \cdot m^2}{deg \cdot s}\) 단위입니다. 라디안을 도(degree)로 변환하려면 다음 수식을 사용할 수 있습니다:

\[S_{deg} = S_{rad} \times \frac{\pi}{180}\]
\[D_{deg} = D_{rad} \times \frac{\pi}{180}\]

effort_limit는 조인트에 가할 수 있는 최대 힘입니다. 해당 값을 조인트 드라이브 API의 Max Force 속성에 설정합니다.

Advanced 탭의 Raw USD Properties로 스크롤하고 환경 정의 파일에 지정된 값으로 Armature와 Joint Friction 속성을 설정합니다.

Maximum Joint Velocity 속성에는 환경 정의 파일에 지정된 velocity_limit 값으로 설정합니다. 도(degree)로 변환하는 것을 잊지 마세요.

\[\omega_{deg} = \omega_{rad} \times \frac{180}{\pi}\]

Note

팔과 다리 같은 로봇의 모든 활성 조인트에 대해 조인트 구성을 설정하는 것을 잊지 마세요.

Verifying the Joint Configuration#

조인트 구성을 확인하려면 시뮬레이션을 실행하고 Script Editor에서 다음 스니펫을 실행하여 조인트 구성을 출력할 수 있습니다.

  1. 시뮬레이션을 실행합니다.

  2. Window > Script Editor를 클릭하여 Script Editor를 엽니다.

  3. 다음 스니펫을 Script Editor에 복사하여 붙여넣습니다.

  4. Run 버튼을 클릭하여 스니펫을 실행합니다.

    from isaacsim.core.experimental.prims import Articulation

prim = Articulation("/h1")
print(prim.dof_names)
lower, upper = prim.get_dof_limits()
stiffnesses, dampings = prim.get_dof_gains()
max_velocities = prim.get_dof_max_velocities()
max_efforts = prim.get_dof_max_efforts()
for i, name in enumerate(prim.dof_names):
    print(
        f"  {name}: lower={lower.numpy()[0][i]:.4f}, upper={upper.numpy()[0][i]:.4f}, "
        f"maxVelocity={max_velocities.numpy()[0][i]:.2f}, maxEffort={max_efforts.numpy()[0][i]:.0f}, "
        f"stiffness={stiffnesses.numpy()[0][i]:.2f}, damping={dampings.numpy()[0][i]:.2f}"
    )

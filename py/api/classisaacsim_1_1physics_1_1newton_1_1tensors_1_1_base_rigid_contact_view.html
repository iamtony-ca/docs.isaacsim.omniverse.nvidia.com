BaseRigidContactView#

Fully qualified name: isaacsim::physics::newton::tensors::BaseRigidContactView

class BaseRigidContactView : public IRigidContactView#

Base contact view implementing IRigidContactView for Newton.

Builds sensor-to-body and filter-to-body mapping tables from USD paths. Caches contact data pointers (forces, normals, shape indices, points) from the Newton contacts object, refreshing lazily when the simulation timestamp changes. Pre-allocates scratch buffers for contact counting and indexing.

Subclassed by isaacsim::physics::newton::tensors::CpuRigidContactView, isaacsim::physics::newton::tensors::GpuRigidContactView

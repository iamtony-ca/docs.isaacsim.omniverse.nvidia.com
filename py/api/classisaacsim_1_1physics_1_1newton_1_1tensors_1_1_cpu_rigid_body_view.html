CpuRigidBodyView#

Fully qualified name: isaacsim::physics::newton::tensors::CpuRigidBodyView

class CpuRigidBodyView : public isaacsim::physics::newton::tensors::BaseRigidBodyView#

CPU rigid body view. Implements getters/setters using host-side gather/scatter loops.

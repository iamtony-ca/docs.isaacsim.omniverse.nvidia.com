BaseSimulationView#

Fully qualified name: isaacsim::physics::newton::tensors::BaseSimulationView

class BaseSimulationView : public ISimulationView#

Base simulation view implementing ISimulationView for the Newton backend.

Holds the Python Newton stage and model references, implements shared logic for step, gravity, object-type queries, and USD pattern matching. Factory methods for child views (articulation, rigid body, rigid contact) resolve USD patterns and delegate to pure virtual make*View() methods overridden by Cpu/GpuSimulationView.

Subclassed by isaacsim::physics::newton::tensors::CpuSimulationView, isaacsim::physics::newton::tensors::GpuSimulationView

UCXSubscribeJointCommandNodeBase#

Fully qualified name: isaacsim::ucx::nodes::UCXSubscribeJointCommandNodeBase

template<typename DatabaseT>
class UCXSubscribeJointCommandNodeBase : public isaacsim::ucx::nodes::UcxNode#

Base class template for UCX joint state subscriber nodes.

Provides common functionality for receiving joint command data via UCX. Derived classes implement message parsing via parseMessage(). The base class handles setting outputs via setOutputs().

Template Parameters:

DatabaseT – The OmniGraph database type for the derived node

Validates port number is in acceptable range for OmniGraph nodes.
